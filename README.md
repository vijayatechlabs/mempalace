# MemPalace Hub (MemPalace.in)

This repository contains the source code for [MemPalace.in](https://www.mempalace.in), the independent resource hub and MCP integration guide for the MemPalace open-source AI memory project.

## Features
- **Live GitHub Stars**: Real-time stats from the core repository.
- **MCP Tool Hub**: Quick-reference guide for all 19 MemPalace tools.
- **Token Savings Calculator**: Interactive ROI projection for local vs cloud context.
- **Palace Visualizer**: Architectural mapping of Wings, Rooms, Closets, and Drawers.
- **Zero-Build Architecture**: Vanilla HTML/Tailwind/JS for <1s load times.
- **GEO Optimized**: Full JSON-LD schema for Generative Engine visibility.

## Tech Stack
- **Structure**: HTML5 (Semantic)
- **Styling**: Tailwind CSS (CDN)
- **Logic**: Vanilla JavaScript
- **Deployment**: GitHub Pages

## Local Development
Since this is a static site with no build step, you can simply open `index.html` in any browser or use a live server extension.

```bash
# Clone the repo
git clone https://github.com/vijayatechlabs/mempalace.git

# No install steps required. Open index.html.
```
