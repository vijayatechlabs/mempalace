Why a Hollywood Actress Started Coding AI Tools

Milla Jovovich (Alice in Resident Evil) noticed a glaring issue with LLMs: every new session suffered from amnesia. Existing memory tools used AI to decide what to forget, discarding the very reasoning she needed most.

Partnering with developer Ben Sigman, they built MemPalace. Their radical approach? Store everything verbatim and make it instantly findable using a "memory palace" metaphor.

"I wanted my AI to remember the way I remember — not just the conclusions, but the journey. The alternatives I considered... That's what existing memory systems throw away."

Frequently Asked Questions

What is MemPalace?
An open-source, local AI memory system that uses a memory palace metaphor (wings, rooms, closets, drawers) to store and retrieve agent conversations.
Does MemPalace require an API key?
No. MemPalace runs 100% locally using ChromaDB and SQLite, requiring zero cloud dependencies.
What is the LongMemEval score for MemPalace?
In raw verbatim mode, MemPalace scores 96.6% R@5 on 500 questions, independent of LLM APIs.
How do I connect MemPalace to Claude?
Run claude plugin marketplace add milla-jovovich/mempalace in your terminal.
